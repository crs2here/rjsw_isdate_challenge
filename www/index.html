Riverside JS Workshop isDate() Code Challenge

Use this UI to test your custom parser method. Both jQuery and Moment.js have been included for your use. Code your solution using the stubbed out method, api.customDateParser(), in the app.js file. Before you begin, please read the isDateChallenge.pdf file. You'll find it in the project root. It will give a better understanding of the requirements for this challenge. When you have a working solution to share, post a link in the Issues section of the RiversideJSW repository. Good luck, and happy coding!
